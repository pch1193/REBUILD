상태 확인 중…

1 · 동기화 코드

두 기기에 같은 코드를 넣으면 기록이 연결돼요. 비밀번호처럼 다루세요. 한 번 만들고 다른 기기에 그대로 입력하면 됩니다.

2 · Firebase 연결 키

Firebase 콘솔에서 복사한 설정 코드를 통째로 붙여넣으세요. { apiKey: … } 형태 그대로면 돼요.

AI 코치 (Gemini)

Google AI Studio(aistudio.google.com/apikey)에서 무료 Gemini API 키를 만들어 붙여넣으면, "AI 코치" 탭에서 내 기록을 바탕으로 질문할 수 있어요. 키는 이 기기에만 저장돼요.

백업

가끔 파일로 저장해두면 안전해요. 다른 기기로 옮기거나 복구할 때 불러오기를 쓰세요.

Firebase 처음 설정하는 법
  1. console.firebase.google.com 접속 → "프로젝트 만들기" (이름 아무거나, 애널리틱스는 꺼도 됨)
  2. 왼쪽 빌드 → Firestore Database → "데이터베이스 만들기" → 위치 선택 → 프로덕션 모드로 시작
  3. 빌드 → Authentication → "시작하기" → 익명(Anonymous) 사용 설정
  4. 프로젝트 설정(⚙ 톱니) → 아래 내 앱에서 웹 앱(</>) 추가 → 닉네임 입력 → 나오는 firebaseConfig 복사 → 위 2번 칸에 붙여넣기
  5. Firestore 규칙(Rules) 탭에 아래를 붙여넣고 게시:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /rebuild/{code}/days/{day} {
      allow read, write: if request.auth != null;
    }
  }
}
